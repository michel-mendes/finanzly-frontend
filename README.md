<p id="description">O Finanzly é um gerenciador de finanças voltado para uso pessoal. Trata-se de uma aplicação multiusuário que permite o registro de uma ou mais carteiras para o controle de transações, organizadas e separadas por categorias de custo.</p>

<h2>🧐 Funcionalidades</h2>

Aqui estão algumas das principais funcionalidades implementadas no projeto:

* Autenticação e login de usuário
* Dashboard financeiro informativo
* Criação e gerenciamento de múltiplas carteiras
* Criação de categorias personalizadas
* Registro e controle de transações
* Importação de transações em lote via arquivo CSV
